# Turtle Grid Game 

A simple grid-based treasure hunt game built with Python and Turtle Graphics.

## How to Play

Start in the bottom-left corner of the grid and use the WASD keys to move around.

A hidden target is randomly placed somewhere on the board. Find it before you run out of moves!

## Controls

* **W** → Move Up
* **A** → Move Left
* **S** → Move Down
* **D** → Move Right

## Features

* Random hidden target location
* Move counter
* 10-move limit
* Win and lose conditions
* Turtle Graphics interface

## Built With

* Python
* Turtle Graphics
* Random Module
